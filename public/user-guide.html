Getting Started

Welcome to Project Zero! This guide will help you get familiar with all the features and functionalities of our document management platform.

Key Features

  • Document Analysis - Upload and analyze legal documents for insights, risks, and opportunities.
  • E-Signature - Create, manage, and apply electronic signatures to documents with multi-party signing workflows.
  • Document Editor - View, edit, annotate, and collaborate on documents including PDFs.
  • Document Comparison - Compare different versions of documents to identify changes.
  • Collaboration - Work simultaneously with others on documents with real-time chat.

Navigation

The main navigation bar at the top of the screen allows you to access the primary features:

  • Document Analysis - Upload and analyze documents
  • E-Signature - Sign documents and manage signature requests
  • Document Editor - Edit, annotate, and collaborate on documents
  • User Guide - Access this comprehensive guide

Document Upload

Before you can analyze, edit, or sign documents, you'll need to upload them to the platform. Here's how to upload documents:

  1. Access Upload Area

    Navigate to either the Document Analysis page or E-Signature page, where you'll find a dedicated upload area.

  2. Select or Drop Files

    Click the "Browse Files" button to select files from your computer, or simply drag and drop files directly onto the upload area.

  3. Wait for Processing

    Once uploaded, your document will be processed. For large documents, this might take a moment. You'll see a progress indicator during the upload.

  4. View Uploaded Document

    After successful upload, you'll either be redirected to the document view or see a success message.

Supported File Types

Project Zero supports various document formats, including:

  • PDF - Portable Document Format files
  • DOC/DOCX - Microsoft Word documents
  • TXT - Plain text files
  • RTF - Rich Text Format files

The maximum file size for uploads is 25MB.

Troubleshooting Upload Issues

If you encounter problems when uploading documents:

  • Ensure your file is in a supported format
  • Check that the file size is under 25MB
  • Try using a different browser or clearing your browser cache
  • If the upload progress gets stuck, try refreshing the page and uploading again
  • Make sure you have a stable internet connection

Document Analysis

The Document Analysis feature helps you extract insights from your legal documents, identify potential risks, and provide recommendations.

  1. Upload a Document

    Start by uploading a document through the Document Analysis page as described in the previous section.

  2. Analysis Process

    Once uploaded, the document will be automatically processed and analyzed. This process may take a few moments depending on the document size and complexity.

  3. View Analysis Results

    After analysis is complete, you'll be taken to a results page showing key insights, potential risks, and recommendations based on the document content.

  4. Explore Detailed Insights

    Click on any section in the analysis results to see more detailed information about specific clauses, terms, or potential issues.

Analysis Features

  • Key Clause Extraction - Identifies important clauses in the document
  • Risk Assessment - Highlights potential legal or business risks
  • Entity Recognition - Identifies organizations, people, dates, and monetary values
  • Document Summary - Provides a concise overview of the document

Document Editor

The Document Editor provides a comprehensive set of tools for viewing, editing, annotating, and collaborating on documents.

  1. Access the Editor

    Click on "Document Editor" in the main navigation or select "Edit" from the document actions menu after uploading.

  2. Select a Document

    If you haven't already selected a document, you'll be prompted to choose one from your uploaded documents or upload a new one.

  3. Explore the Interface

    The editor interface consists of:

    • Left sidebar with tools
    • Main document viewing area
    • Top toolbar with contextual controls
    • Optional collaboration panel (right side)

  4. Use Document Tools

    Select different tools from the sidebar to view, edit, annotate, sign, compare, or collaborate on the document.

Navigation Controls

  • Zoom controls - Zoom in/out of the document
  • Page controls - Navigate between pages of multi-page documents
  • Document info - View and click to change the current document
  • Save button - Save your changes

Annotations

The annotation tools allow you to add notes, comments, and highlights to documents without changing the original content.

  1. Select Annotate Tool

    Click the "Annotate" tool in the left sidebar of the Document Editor.

  2. Choose Annotation Type

    Select either "Highlight" or "Comment" from the toolbar that appears at the top.

  3. Create Annotation

    For highlights or comments, click and drag across the text or area you want to annotate.

  4. Add Comment Text

    If adding a comment, a dialog will appear where you can enter your comment text.

  5. View and Edit Annotations

    Click on any annotation to view its content. If you're the creator, you can edit or delete it.

Annotation Tips

  • Annotations are saved with the document and visible to collaborators
  • Use different colors for different types of annotations (coming soon)
  • Annotations can be filtered or hidden using the view options
  • You can export a document with or without annotations

Document Editing

While PDF editing requires a premium PDF library, you can still make basic changes to documents:

  1. Select Edit Tool

    Click the "Edit" tool in the left sidebar of the Document Editor.

  2. Choose Edit Type

    Select either "Text" or "Image" from the toolbar that appears at the top.

  3. Make Changes

    For full document editing capabilities, consider upgrading to the premium version.

Note: Full document editing is available in the premium version, which includes the ability to edit text, add images, and make other changes directly in PDFs and other document formats.

E-Signature

The E-Signature feature allows you to create, manage, and apply electronic signatures to documents with legally-binding validity.

Creating Signatures

There are three ways to create your signature:

  1. Draw Your Signature

    Use your mouse or touchscreen to draw your signature directly.

  2. Type Your Signature

    Type your name and select a signature style to create a typed signature.

  3. Upload a Signature Image

    Upload an image of your handwritten signature.

Signing Documents

  1. Upload a Document

    Start by uploading the document you need signed through the E-Signature page.

  2. Add Signature Points

    In the Document Editor, select the "Sign" tool from the sidebar. Click where you want to add a signature in the document.

  3. Choose Signature Method

    Select your preferred signature method (draw, type, or upload) when the signature dialog appears.

  4. Apply Your Signature

    Create your signature and click "Apply Signature" to place it on the document.

  5. Complete Signing

    After adding all required signatures, click "Save" to save the signed document.

Multiple Signers

  1. Add Signers

    While viewing your document in the Document Editor, select the "Collaborate" tool and click "Invite" to add signers.

  2. Enter Signer Details

    Enter the name and email address of each person who needs to sign the document. You can also add a personalized message.

  3. Assign Signature Fields

    Use the "Sign" tool to place signature fields for each signer. You can specify which signer each field is for.

  4. Send for Signature

    Once you've added all signers and assigned fields, save the document to notify them via email.

  5. Track Progress

    You can track the signing progress in the document details page, which shows who has signed and who still needs to sign.

E-Signature Security

All signatures are:

  • Time-stamped with creation date
  • Associated with the signer's identity
  • Securely stored and tamper-evident
  • Compliant with major e-signature laws

Document Comparison

The Document Comparison feature allows you to compare different versions of a document to identify changes.

  1. Access Comparison Tool

    In the Document Editor, select the "Compare" tool from the sidebar.

  2. Select Documents

    Choose the second document you want to compare with the current document.

  3. Choose Comparison Type

    Select either "Text comparison" (to detect changes in content) or "Visual comparison" (for side-by-side view).

  4. View Results

    After comparison is complete, the differences between the documents will be highlighted.

Comparison Features

  • Text Comparison - Highlights added, removed, and changed text
  • Visual Comparison - Shows documents side by side for visual reference
  • Change Summary - Provides a list of all changes detected

Collaboration

The Collaboration features allow you to work with others on the same document in real-time.

  1. Invite Collaborators

    In the Document Editor, select the "Collaborate" tool from the sidebar and click "Invite".

  2. Set Permissions

    Enter the email addresses of your collaborators and set their permissions:

    • Viewer - Can only view the document
    • Commenter - Can view and add comments
    • Editor - Can make changes to the document
    • Owner - Has full control including inviting others

  3. Send Invitations

    Add an optional message and send the invitations. Collaborators will receive an email with a link to join.

  4. Use Collaboration Panel

    Click the collaborators icon in the top toolbar to open the collaboration panel, where you can see who's viewing the document and chat with them.

Real-time Features

  • Presence Indicators - See who is currently viewing the document
  • Chat - Communicate with other collaborators in real-time
  • Change Tracking - See who made what changes to the document

Troubleshooting

If you encounter issues while using Project Zero, try these solutions to common problems:

Upload Issues

  • File too large - Ensure your file is under 25MB
  • Unsupported format - Check that your file type is supported
  • Upload stuck - Refresh the page and try again

Viewing Issues

  • Document not loading - Try refreshing the page or using a different browser
  • Missing pages - Check that the document was completely uploaded
  • Formatting issues - Some complex formatting may not be preserved in all file types

E-Signature Issues

  • Can't add signature - Ensure you're in "Sign" mode
  • Signature not showing - Try refreshing the page
  • Email invites not received - Check spam folders or try resending
